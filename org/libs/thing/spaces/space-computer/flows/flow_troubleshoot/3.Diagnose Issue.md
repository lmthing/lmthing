# Diagnose Issue

Match symptoms and log findings to known root causes:

- **Expired subscription** → Machine can't be accessed, token issuance fails. Check billing status.
- **Resource exhaustion** → CPU at 100%, memory > 90%, disk full. Processes are slow or killed.
- **Token expiry** → Terminal disconnects after ~5 minutes. Dashboard should auto-refresh tokens.
- **Volume full** → Disk at 100%, writes fail, processes crash. Clear files to free space.
- **Network issues** → Can't reach external services, DNS resolution fails. Check Fly.io status.
- **Process crash** → Specific service stopped running. Check logs for crash reason, restart the process.
- **Health check timeout** → Machine is too slow to respond within 5 seconds. Usually caused by resource exhaustion.

Start with the simplest explanation and work up to more complex causes. Most issues are either subscription-related or resource exhaustion.
