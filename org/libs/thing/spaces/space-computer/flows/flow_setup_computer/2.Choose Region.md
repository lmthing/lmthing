# Choose Region

Help the user select the best Fly.io region for their computer. The choice affects latency and data location.

**Available regions:**
- **IAD** (US East) — Default. Best for Americas, lowest inter-service latency.
- **LHR** (London) — Best for UK/Western Europe, GDPR considerations.
- **AMS** (Amsterdam) — Best for Central Europe, excellent peering.
- **SJC** (US West) — Best for West Coast, Asia-Pacific connectivity.

Consider the user's physical location, where their users are, and any data sovereignty requirements. When in doubt, recommend IAD — it's the default and has the best connectivity to other lmthing services.
