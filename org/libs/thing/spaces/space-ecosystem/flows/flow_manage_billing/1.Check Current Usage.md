# Check Current Usage

Query the user's current Stripe balance via the get-usage cloud endpoint. This returns the customer's balance in cents.

Explain what the numbers mean:
- **Negative balance** = credit remaining (prepaid amount not yet consumed)
- **Positive balance** = amount owed (usage beyond prepaid credit)
- **Zero balance** = no credit and no outstanding charges

Help the user understand their token consumption patterns. Each LLM request is automatically metered by Stripe — input tokens and output tokens are counted separately, and pricing varies by model (premium models cost more per token than basic ones).

If the user is on the free tier and has exhausted their budget, explain their options: wait for the next billing period or upgrade to Pro for higher limits.
