# Improve Description

Rewrite or refine the description based on your review and competitive analysis.

Focus on these improvements:
- **Strengthen the hook** — Make the first line impossible to skip
- **Clarify the value proposition** — Be specific about outcomes, not just capabilities
- **Add concrete examples** — Show exactly what the agent produces
- **Improve scannability** — Use bullet points, bold key phrases, short paragraphs
- **Sharpen the differentiator** — Why this agent over alternatives?

If there's no preview or demo, add one. This is often the single highest-impact change you can make. A screenshot of a great conversation or a recorded demo session can dramatically increase conversion.

Update documentation if it's incomplete — add missing getting-started steps, configuration guides, or example prompts.
