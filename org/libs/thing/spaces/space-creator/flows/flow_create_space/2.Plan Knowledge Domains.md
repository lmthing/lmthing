- **Layered Context** — Best for complex, multi-layered subjects. Splits from broad environment down to narrow specifics.

- A kebab-case folder name (e.g., `user-context`)

- A one-sentence purpose statement

You MUST include at least one context domain (like `user-context` or `creator-context`) that captures user-specific information the agent needs at runtime.
