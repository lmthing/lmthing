# Review Plan Options

Compare the three subscription tiers to help the user understand their options:

**Free** — No cost. Basic models (gpt-4o-mini, claude-3-5-haiku). Limited monthly token budget. Full Studio and Chat access. Good for exploring and learning.

**Pro** — Full model catalog including GPT-4o, Claude Opus, Gemini Pro. Higher token limits. Full Blog access. Store publishing. For regular agent builders and daily users.

**Computer** — Everything in Pro plus a dedicated Fly.io machine (1 CPU, 1 GB RAM, terminal access). For power users needing persistent runtimes, autonomous agents, or cloud development environments.

Help the user identify which tier matches their needs. Most users start on Free and upgrade to Pro when they need premium models or hit token limits. Computer is for users who need infrastructure.
