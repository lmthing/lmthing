# Premium Models

The full model catalog available on Pro and Computer tier subscriptions. Includes flagship models from OpenAI, Anthropic, and Google.

## Available Models

**OpenAI:**
- **openai/gpt-4o** — Flagship multimodal. Strong general capability, reliable tool calling, fast.
- **openai/gpt-4.1** — Latest generation. Improved instruction following and coding.
- **openai/o3** — Reasoning model. Slower but more thoughtful, step-by-step analysis.

**Anthropic:**
- **anthropic/claude-opus-4** — Flagship. Exceptional reasoning, nuanced writing, complex tasks.
- **anthropic/claude-sonnet-4** — Balanced. Strong quality with better speed than Opus.

**Google:**
- **google/gemini-2.5-pro** — Large context window. Great for long documents and research.
- **google/gemini-2.5-flash** — Fast and efficient. Good for high-volume tasks.

## Choosing the Right Model

- **General tasks** → GPT-4o (fast, reliable, versatile)
- **Quality writing** → Claude Sonnet or Opus (nuanced, careful)
- **Complex reasoning** → o3 or Claude Opus (methodical, thorough)
- **Long documents** → Gemini Pro (massive context window)
- **Speed-critical** → GPT-4o or Gemini Flash (fast response times)
- **Code tasks** → GPT-4.1 or Claude Sonnet (strong code generation)

## Billing

All premium models are metered through Stripe Token Billing. Costs vary by model — flagship models (Opus, o3) cost more per token than mid-range models (Sonnet, GPT-4o). Check your usage via the billing portal or get-usage endpoint.
