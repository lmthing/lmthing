# Select Dish Category

Identify the type of dish you want to create. Greek cuisine offers diverse categories:

- **Meze (Μεζέδες)**: Small plates for sharing - dips, cheese, olives
- **Soups (Σούπες)**: Avgolemono, fasolada, trachanas
- **Main Courses (Κύρια Πιάτα)**: Moussaka, pastitsio, grilled meats
- **Seafood (Θαλασσινά)**: Fresh fish, octopus, shrimp saganaki
- **Vegetarian (Λαδερά)**: Olive oil-based vegetable dishes
- **Pies (Πίτες)**: Spanakopita, tiropita, regional variations
- **Desserts (Γλυκά)**: Baklava, galaktoboureko, kourabiedes

Consider the occasion and number of servings needed.
