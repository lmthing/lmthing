# Begin Conversation

Start the conversation with a clear, effective opening message.

**Crafting your first message:**
- State your goal or question directly — don't start with "Hi, can you help me with..."
- Include the most important context in the first message
- If you have a specific output format in mind, state it upfront

**Steering the conversation:**
- If responses go off track, restate your goal and add constraints
- Switch models mid-conversation if the current one isn't performing well
- Change modes if the interaction style doesn't match your needs
- Use follow-up questions to drill deeper into specific aspects

**Managing long conversations:**
- Context accumulates over a conversation — earlier messages influence later responses
- If the conversation drifts, start a new one with refined context
- Save important outputs before they scroll out of view
- Consider starting fresh rather than trying to correct a long-drifted conversation
