# Choose Your Starting Point

Recommend the best service to start with based on the user's goal:

- **Most users** → Start with **Studio**. Build your first agent, explore the workspace, test conversations. This is the natural entry point.
- **Developers** → Start with **com/** for API keys, then use the generate-ai endpoint directly. Skip the UI if you prefer code.
- **Quick conversations** → Start with **Chat**. No setup needed — just pick a model and start talking.
- **Home automation** → Start with **Casa** if you have Home Assistant. You'll need a Computer tier subscription.

For each recommendation, provide the concrete next step: the URL to visit, the first action to take, and what they'll see when they get there. Remove all ambiguity about what to do next.
