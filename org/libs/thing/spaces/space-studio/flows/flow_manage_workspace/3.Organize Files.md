# Organize Files

Move or rename files to follow lmthing naming conventions:

- **Folders** — kebab-case (agent-formula-expert, knowledge-design)
- **Agent names** — PascalCase in instruct.md frontmatter (FormulaExpert)
- **Flow IDs** — flow_snake_case for folders (flow_create_space)
- **Flow steps** — Numbered Title Case files (1.Define Your Subject.md)
- **Variables** — camelCase in field config.json (experienceLevel)
- **Domain references** — domain- prefix in enabledKnowledgeFields (domain-space-structure)

Check for common naming errors: mixed case in folder names, missing domain- prefix, inconsistent flow naming. Fix these before they cause runtime errors.
