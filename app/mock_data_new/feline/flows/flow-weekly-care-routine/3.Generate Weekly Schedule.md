# Step 3: Generate Weekly Schedule

**Agent:** Cat Care Advisor

## Task

Create a weekly schedule for less-frequent care tasks.

## Inputs

- Daily checklist from Step 2
- Grooming frequency requirements
- Cat's personality and tolerance for handling
- Owner's schedule flexibility

## Instructions

Organize weekly tasks by day:

### Suggested Weekly Distribution

**Monday:**
- Deep litter box cleaning
- Weigh cat (if monitoring weight)

**Wednesday:**
- Grooming session (brushing, nail check)
- Inspect ears and teeth

**Friday:**
- Play area rotation/refresh
- Clean food/water bowls thoroughly
- Treat puzzle or special enrichment

**Saturday/Sunday:**
- Extended play/bonding session
- Window perch cleaning
- Toy rotation
- Any grooming tasks that need more time

### Adjust based on:
- Cat's tolerance (spread tasks if cat gets stressed)
- Owner's schedule (consolidate if limited time on weekdays)
- Specific needs (more frequent grooming for long-haired cats)

## Output

A weekly calendar with specific tasks assigned to optimal days, with rationale for the distribution.
