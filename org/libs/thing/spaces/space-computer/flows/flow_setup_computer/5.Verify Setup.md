Run a series of verification checks to confirm the computer is fully operational:

- **Health endpoint** — Verify /health responds with 200
- **Volume** — Check the volume is mounted with `df -h`
- **Environment** — Confirm environment variables are set: USER_ID, RUNTIME_MODE
- **Shell** — Run basic commands to verify terminal works: `pwd`, `ls`, `node --version`
- **Network** — Test outbound connectivity: `curl -s https://httpbin.org/ip`

Report the result of each check. If any check fails, note the specific error and recommend using the /diagnose flow to troubleshoot. Once all checks pass, confirm the computer is ready for use as a persistent cloud development environment.
