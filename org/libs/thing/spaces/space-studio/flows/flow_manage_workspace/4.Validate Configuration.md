# Validate Configuration

Cross-check all references to ensure the workspace is internally consistent:

- **flowIds** — Every flowId in agent slash_action tags must match an existing flow directory name
- **enabledKnowledgeFields** — Every entry must reference an existing domain (domain-{folder-name})
- **runtimeFields** — Every domain and field slug must exist in the knowledge/ directory
- **Defaults** — Every field config.json default must match an option file slug
- **Package.json** — Every space must have name, version, and private: true

Report any broken references with specific file paths and suggested fixes. Validation should be exhaustive — a single broken reference can cause runtime errors.
