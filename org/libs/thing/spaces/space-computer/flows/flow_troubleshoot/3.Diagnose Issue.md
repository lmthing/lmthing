Correlate the symptom report and log analysis to determine the root cause. Match against these known issue patterns:

- **Expired subscription** — Machine inaccessible, token issuance fails. Verify billing status.
- **Resource exhaustion** — CPU at 100%, memory > 90%, disk full. Processes slow or killed.
- **Token expiry** — Terminal disconnects after ~5 minutes. Dashboard should auto-refresh tokens.
- **Volume full** — Disk at 100%, writes fail, processes crash. Need to clear files.
- **Network issues** — Cannot reach external services, DNS resolution fails. Check Fly.io status.
- **Process crash** — Specific service stopped running. Check logs for crash reason.
- **Health check timeout** — Machine too slow to respond within 5 seconds. Usually resource exhaustion.

Start with the simplest explanation and work up to more complex causes. Most issues are either subscription-related or resource exhaustion. Produce a definitive diagnosis with confidence level.
