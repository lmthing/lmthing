# Create Cooking Timeline

Plan backwards from serving time to ensure everything is ready:

## Day Before
- [ ] Shop for all ingredients
- [ ] Make dips (tzatziki, melitzanosalata) — flavors improve overnight
- [ ] Prepare dolmades
- [ ] Marinate meat
- [ ] Bake desserts (baklava, galaktoboureko)
- [ ] Make syrup for desserts

## Morning Of
- [ ] Prep all vegetables (wash, chop)
- [ ] Bring meat to room temperature
- [ ] Start any slow-cooking dishes (stews, roasts)
- [ ] Prepare stuffed vegetables
- [ ] Set the table

## 2 Hours Before
- [ ] Start oven dishes (potatoes, yemista)
- [ ] Prepare salad ingredients (don't dress yet)
- [ ] Slice bread
- [ ] Arrange cold meze on platters

## 1 Hour Before
- [ ] Start rice/pasta
- [ ] Cook hot meze
- [ ] Grill meats
- [ ] Warm dips if needed

## 30 Minutes Before
- [ ] Dress salads
- [ ] Final seasoning on dishes
- [ ] Plate and arrange
- [ ] Pour drinks for arrival

## Key Principles
- Don't stress about timing — Greek time is flexible
- Better to have food waiting than guests waiting
- Let guests snack on meze while you finish
- Accept help in the kitchen — it's social
