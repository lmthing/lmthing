# Step 3: Symptom Tracking System

**Agent:** Vet Visit Planner

## Task

Create a symptom tracking system for when health concerns arise.

## Instructions

Design tracking tools for common symptoms:

### Daily Symptom Log Template

**Date:** _______

**Symptom observed:** _______________________________

**Time first noticed:** _______

**Frequency today:** _______

**Severity (1-10):** _____

**Duration:** _______

**Associated behaviors:**
- Eating: Normal / Reduced / None
- Drinking: Normal / Increased / Decreased
- Litter box: Normal / Increased / Decreased / Straining / Blood
- Energy: Normal / Lethargic / Hyperactive
- Mood: Normal / Hiding / Aggressive / Clingy

**What makes it better:** _______________________________

**What makes it worse:** _______________________________

**Other observations:** _______________________________

### Symptom-Specific Tracking

**Vomiting/Diarrhea:**
- Time of each occurrence
- Appearance (color, consistency, content)
- Relation to meals
- Other symptoms present

**Appetite Changes:**
- Amount eaten (% of normal)
- Foods accepted/refused
- Treats accepted?
- Weight change

**Litter Box Issues:**
- Number of times per day
- Amount
- Appearance
- Straining?
- Location (in box or outside?)

**Respiratory:**
- Breathing rate (breaths per minute at rest)
- Coughing frequency
- Sneezing frequency
- Nasal discharge
- Open-mouth breathing?

**Pain Indicators:**
- Location (if apparent)
- When does cat show discomfort?
- Position changes
- Vocalization
- Aggression when touched

### When to Call Vet

Provide decision tree based on symptom and duration:

**Call immediately (emergency):**
- Difficulty breathing
- Unconscious/unresponsive
- Seizure
- Suspected poison
- Trauma
- Bloated/hard abdomen
- Can't urinate (especially males)

**Call within 24 hours:**
- Vomiting/diarrhea > 24 hours
- Not eating > 24 hours
- Lethargy
- Signs of pain
- Limping
- Eye injury

**Monitor and call if persists 48-72 hours:**
- Mild increased drinking
- Slight appetite decrease
- Minor behavior change
- Occasional vomiting (hairballs)

## Output

- Daily symptom log (printable/fillable)
- Symptom-specific tracking sheets
- Vet call decision flowchart
- Video/photo taking guidance (for showing vet)
