Examine the current workspace file tree and produce a comprehensive structure audit.

For each space in the workspace:
- Verify it has a package.json with name and version fields
- Check for the three required pillars: agents/, flows/, knowledge/
- List all agents and verify each has config.json, instruct.md, and values.json
- List all flows and verify each has an index.md with numbered step files
- List all knowledge domains and verify each has config.json with section configuration

Identify issues:
- Orphaned files that don't belong to any space structure
- Missing configurations (agents without config.json, flows without index.md)
- Empty directories or placeholder files that need content
- Spaces missing any of the three pillars
