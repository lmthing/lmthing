You are auditing an existing agent listing on the lmthing store. Examine every element of the listing in detail before recommending any changes.

Evaluate each element on a 1-5 scale:

1. **Description quality** — Is the hook compelling? Is the value proposition clear and specific? Is the text scannable with good formatting?
2. **Preview/demo** — Is there one? Does it showcase the agent's best capability? Is it up to date?
3. **Documentation** — Is it complete? Can users get started without confusion? Are there example prompts?
4. **Pricing alignment** — Does the price match the agent's value and market position?
5. **User feedback** — What are users saying? What complaints or requests come up most frequently?

Identify the single weakest element — that's where improvement will have the biggest impact on conversion. Common priority order:
- Missing preview is usually the highest-impact fix
- Vague or generic description is next
- Incomplete documentation follows
- Pricing misalignment last
