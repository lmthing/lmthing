# Attach Flows to Agents

Flows give agents the ability to run multi-step processes via slash commands.

On the Agent Configuration page, open the **Actions** tab in the right panel and click **Attach Flow**.

This opens the "Attach Workflow" modal which lets you:
- Browse available flows in the workspace
- See each flow's name, description, and step count
- Select and attach a flow

Once attached, the action card appears showing:
- The slash command (e.g., `/generate`)
- The flow name
- Number of steps
- Active status badge (green)

**Managing attached flows:**
- Hover over a card to reveal the **edit / disable / detach** menu
- Click the status badge to toggle Active ↔ Inactive without losing the attachment
- Detach removes the action from the agent

**Note:** Flows must exist in the workspace before they can be attached. You create flows by adding markdown files to `flows/flow_*/` directories.
