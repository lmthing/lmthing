# Identify Symptoms

Gather information about what's going wrong. Ask specific questions:

- **Is the machine reachable?** Can you access the dashboard? Does the health endpoint respond?
- **Is it slow?** Are commands taking longer than usual? Is the terminal laggy?
- **Are commands failing?** What error messages are you seeing? Which commands work and which don't?
- **Is the terminal disconnecting?** How frequently? After how long? Does reconnecting work?
- **Did something change recently?** New packages installed? Running a heavy process? Configuration changes?

Understanding the symptom pattern helps narrow down the root cause quickly. Multiple symptoms often point to a single underlying issue — resource exhaustion, for example, causes slowness, command failures, and disconnections simultaneously.
