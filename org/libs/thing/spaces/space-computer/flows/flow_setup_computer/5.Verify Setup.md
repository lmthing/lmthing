# Verify Setup

Confirm the computer is fully operational by running basic checks:

- **Health endpoint** — Verify /health responds with 200
- **Volume** — Check the volume is mounted with `df -h`
- **Environment** — Confirm environment variables are set: `echo $USER_ID`, `echo $RUNTIME_MODE`
- **Shell** — Run a few basic commands to verify the terminal works: `pwd`, `ls`, `node --version`
- **Network** — Test outbound connectivity: `curl -s https://httpbin.org/ip`

If any check fails, note the specific error and use the `/diagnose` flow to troubleshoot. Once everything passes, the computer is ready for use — you can run agents, manage files, and use it as your persistent cloud development environment.
