# Define Agent Role

Clarify what the agent specializes in and establish its identity:

- **Name** — PascalCase (e.g., FormulaExpert, DataAnalyst). Must correspond to a kebab-case folder (agent-formula-expert).
- **Description** — One sentence explaining the agent's expertise and purpose.
- **Pattern** — Choose: specialist (focused expert, most common), coordinator (orchestrates other agents), or reviewer (validates and reviews outputs).
- **Boundaries** — What the agent should and should NOT do. Clear boundaries prevent role drift.

Determine the agent's relationship to other agents in the space. Each agent should have a distinct, non-overlapping role. If two agents overlap in expertise, merge them or sharpen their boundaries.
