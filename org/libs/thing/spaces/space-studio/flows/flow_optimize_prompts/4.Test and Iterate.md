# Test and Iterate

Run the updated agent through test conversations and compare before/after behavior:

- Use the same problematic prompts that triggered the original issue
- Test edge cases — questions at the boundary of the agent's expertise
- Verify the fix didn't introduce new problems (e.g., over-constraining made the agent refuse valid questions)
- Check that knowledge domains are being used appropriately
- Verify tool usage patterns are correct

If the fix worked, document what changed and why in a commit message when pushing to GitHub. If it didn't work, go back to the analysis step and look for a different root cause.

Prompt optimization is iterative — expect 2-3 rounds of refinement for complex agents. Each round should produce measurable improvement on the specific issue being addressed.
