# Step 1: Behavior Assessment

**Agent:** Behavior Trainer

## Task

Thoroughly analyze the target behavior or training goal.

## Inputs

From knowledge base:
- Current behavioral issues
- Cat's personality and temperament
- Training goals
- Living environment
- Socialization history
- Owner's experience level

From user:
- Specific behavior description
- When it occurs
- How often
- Duration/severity
- What has been tried already

## Instructions

Conduct a comprehensive behavior analysis:

### 1. Behavior Description
- Exactly what the cat is doing
- Context and triggers
- Frequency and pattern
- Duration and intensity
- Recent changes or escalation

### 2. Function Analysis
What need does this behavior serve for the cat?
- Attention seeking
- Stress/anxiety relief
- Territorial marking
- Hunting/play drive
- Medical discomfort
- Environmental enrichment lack
- Resource competition (multi-cat)

### 3. Antecedents (What Happens Before)
- Time of day
- Presence of specific people/animals
- Environmental changes
- Activity level
- Food/meal timing

### 4. Consequences (What Happens After)
- How does owner typically respond?
- What does cat get/avoid?
- Is behavior being inadvertently reinforced?

### 5. Risk Assessment
- Is cat or anyone at risk?
- Is this a medical issue requiring vet care?
- Is professional behaviorist needed?

## Output

A detailed behavior analysis that identifies the function of the behavior and potential intervention points.
