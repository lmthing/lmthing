# Review Current Listing

Examine the existing listing in detail before making any changes. Evaluate each element:

- **Description quality** — Is the hook compelling? Is the value proposition clear? Is it scannable?
- **Preview/demo** — Is there one? Does it showcase the agent's best capability?
- **Documentation** — Is it complete? Can users get started without confusion?
- **Pricing** — Does it match the agent's value and market position?
- **User feedback** — What are users saying? What complaints or requests come up?

Identify the weakest element — that's where improvement will have the biggest impact on conversion. A missing preview is usually the highest-impact fix, followed by a vague description, then incomplete documentation.
