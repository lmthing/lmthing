# Listing Description

A compelling description is the most important element of your store listing. It's the first thing potential users read and the primary factor in their download or purchase decision.

## Structure

Write your description in this order:

1. **Hook** — An attention-grabbing first line that states what the agent does and why it matters
2. **Value proposition** — What problems does it solve? What outcomes does it deliver?
3. **Key features** — Bullet points listing the agent's main capabilities
4. **Target audience** — Who is this agent for? Be specific
5. **Differentiator** — What makes this agent unique compared to alternatives?

## Writing Tips

- Lead with benefits, not features — "Save 2 hours per report" beats "Generates reports"
- Use concrete examples and specific outcomes rather than vague promises
- Keep it scannable — use bullet points, short paragraphs, bold key phrases
- Avoid jargon unless your target audience expects it
- Front-load the most important information — many users only read the first few lines
- Include the agent's name and primary use case in the first sentence (helps with search)

## Common Mistakes

- Being too vague — "A powerful AI assistant" tells users nothing
- Being too technical — describing implementation details instead of benefits
- Being too long — descriptions over 300 words lose readers
- No clear audience — if everyone is the target, no one feels targeted
- Missing the "why" — explaining what it does without explaining why it matters
