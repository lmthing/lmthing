# Identify Your Goal

Start by understanding what the user wants to accomplish with lmthing. Common goals include:

- **Build agents** — Create custom AI agents with specific expertise and workflows
- **Deploy agents** — Put agents into production containers accessible via web or API
- **Sell agents** — Publish agents on the marketplace for others to use
- **Automate tasks** — Use agents for recurring tasks, research, or content generation
- **Collaborate** — Work with teams using shared agents and context
- **Integrate** — Connect lmthing capabilities into existing applications via API

Understanding the goal determines which services matter most and in what order to introduce them. A user who wants to build agents needs Studio first. A developer who wants API access needs com/ and API keys first.
