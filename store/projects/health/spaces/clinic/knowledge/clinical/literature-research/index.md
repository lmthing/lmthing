# Grounding a write-up in the literature

Every `research` row the researcher writes traces back to real, checkable sources — never a
summary composed purely from general knowledge presented as if it were sourced. Two disciplines
make that trustworthy: choosing sources that deserve the trust the user will place in them
(`trusted-sources.md`), and writing the summary itself in a way that's honest about what was found
and where it came from (`citation-standards.md`).

The `sources` table lets the user (or the interpreter, when queuing a dive) flag domains, journals,
or guideline bodies the researcher should weight more heavily via `trust` — check it before an open
web search, and prefer a listed source over an equally-relevant unlisted one when both are
available. The end product should read as something the user could genuinely hand to their own
clinician: concise, sourced, and honest about its own limits.
