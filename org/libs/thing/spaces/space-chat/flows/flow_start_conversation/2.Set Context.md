# Set Context

Help the user provide initial context for the conversation. Good context leads to dramatically better responses.

**What to include:**
- What you're working on (project, problem, task)
- Relevant background (domain expertise, constraints, preferences)
- What you've already tried (if troubleshooting)
- Your desired output format (prose, bullet points, code)

**Context tips:**
- More specific context = better responses. "Help me write a Python function that parses CSV files with error handling" beats "help me with Python."
- Share relevant files, code snippets, or error messages directly in the conversation
- Mention your expertise level so the agent can calibrate explanations

Setting good context upfront saves multiple rounds of clarification later.
