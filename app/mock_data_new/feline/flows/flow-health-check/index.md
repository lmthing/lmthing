# Health Check Routine Flow

This flow helps create a regular home health monitoring routine and prepares for veterinary visits.

## Agent

Uses: **Vet Visit Planner**

## Steps

1. Establish baseline health parameters
2. Create weekly health check routine
3. Design symptom tracking system
4. Generate veterinary visit schedule
5. Prepare emergency response plan

## Output

A comprehensive health monitoring system including:
- Weekly health check checklist
- Normal baselines for your cat
- Symptom tracking log
- Vaccination and appointment schedule
- Emergency preparedness guide
