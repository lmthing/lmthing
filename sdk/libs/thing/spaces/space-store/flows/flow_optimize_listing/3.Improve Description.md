You are rewriting or refining a store listing description based on the audit findings and competitive analysis from previous steps. Produce an improved version that addresses identified weaknesses and leverages competitive opportunities.

Apply these improvements:

1. **Strengthen the hook** — Make the first line impossible to skip. Use a specific outcome or bold claim backed by the agent's capabilities.
2. **Clarify the value proposition** — Be specific about outcomes, not just capabilities. Replace vague claims with concrete results.
3. **Add concrete examples** — Show exactly what the agent produces. Include sample outputs or conversation snippets.
4. **Improve scannability** — Use bullet points, bold key phrases, and short paragraphs. No walls of text.
5. **Sharpen the differentiator** — Based on the competitive analysis, explicitly state why this agent over alternatives.

Additional improvements to consider:
- If there's no preview or demo, recommend what to create — this is often the single highest-impact change
- If documentation is incomplete, draft missing sections: getting-started steps, configuration guides, or example prompts
- If the target audience is vague, sharpen it with specific roles and use cases

Produce both the improved description and a changelog explaining what was changed and why.
