You are evaluating an agent for publishing to the lmthing store. Analyze the agent's completeness, quality, and self-containedness to determine if it is ready for the marketplace.

Assess the following:

1. **File completeness** — Does the agent have config.json, instruct.md, and values.json?
2. **Cross-reference validity** — Do all flowIds match flow directories? Do enabledKnowledgeFields reference existing domains? Do runtimeFields point to real fields?
3. **Conversation reliability** — Has the agent been tested in multi-turn conversations? Does it handle edge cases gracefully?
4. **Self-containedness** — Can this agent function independently, or does it depend on other agents, shared knowledge, or space-level resources that won't be included?
5. **Documentation quality** — Is the instruct.md clear about the agent's role, capabilities, and limitations?

If the agent depends on other agents or shared knowledge not included in the package, flag this as a potential issue for buyers.

Produce a readiness assessment with a clear publish/not-ready verdict.
