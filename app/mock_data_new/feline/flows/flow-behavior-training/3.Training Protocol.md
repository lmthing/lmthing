# Step 3: Training Protocol

**Agent:** Behavior Trainer

## Task

Design the step-by-step training protocol using positive reinforcement.

## Inputs

- Behavior analysis (Step 1)
- Environmental setup plan (Step 2)
- Cat's food motivation and preferences
- Owner's available time and consistency capability

## Instructions

Create a detailed training plan:

### 1. Identify Reinforcers
What motivates this cat?
- Food treats (what type, how high-value)
- Play/toys
- Affection/petting
- Verbal praise
- Access to favorite spots

### 2. Break Down Target Behavior
Divide the end goal into small, achievable steps:
- Start with easiest approximation
- Gradually increase criteria
- Set realistic progression timeline

### 3. Training Session Structure

**Frequency:** 
- Short sessions (3-5 minutes)
- 2-3 times daily
- Always end on success

**Protocol:**
1. Setup/preparation
2. Cue (if applicable)
3. Behavior observation
4. Immediate reinforcement (< 1 second)
5. Brief break
6. Repeat

### 4. Behavior-Specific Plan

For unwanted behaviors:
- Redirect to appropriate alternative
- Reinforce the alternative heavily
- Never punish
- Use timeout only for serious aggression (immediately remove yourself, not cat)

For training new behaviors:
- Capture naturally occurring behavior
- Shape gradually toward goal
- Add cue once behavior is reliable
- Practice in different contexts

### 5. Common Scenarios

**Scratching furniture:**
- Catch approaching scratching post → treat
- Any use of post → jackpot reward
- Scratching furniture → redirect without emotion

**Litter box training:**
- After using box correctly → immediate treat
- Create strong positive association
- Never punish accidents

**Aggression during play:**
- Immediately stop play at first sign of overstimulation
- Redirect to toy
- Resume only when calm
- Teach bite inhibition with interrupter word

**Excessive meowing:**
- Never reinforce demanding meows
- Wait for quiet moment → reward
- Gradually increase quiet duration required
- Ensure needs are otherwise met

## Output

A day-by-day training protocol for the first 2 weeks, including:
- Session timing and frequency
- Specific behaviors to reinforce
- Progression criteria
- What to record/track
