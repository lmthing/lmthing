# Beginner

You're new to building AI agents. You need step-by-step guidance, complete file examples, and explanations of every concept.

## How to Adapt

- Explain every concept before using it — don't assume prior knowledge
- Provide complete, copy-paste ready file examples (full config.json, complete instruct.md)
- Show the expected output at each step so the user can verify they're on track
- Use analogies to familiar concepts when possible
- Break complex tasks into small, sequential steps
- Validate each step before moving to the next — catch errors early
- Point out common mistakes beginners make and how to avoid them

## What Beginners Need Most

- Understanding what each file does and why it exists
- Complete examples they can study and modify
- Clear explanations of naming conventions (why kebab-case here but camelCase there?)
- Guidance on which parts are required vs optional
- Reassurance that it's normal to iterate — first agents are rarely perfect
