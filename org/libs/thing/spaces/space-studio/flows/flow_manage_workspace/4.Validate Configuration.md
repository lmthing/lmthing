Perform exhaustive cross-reference validation across the entire workspace and report all broken or inconsistent references.

Validate each category:
- **flowIds** — every flowId in agent slash_action tags must match an existing flow directory name
- **enabledKnowledgeFields** — every entry must reference an existing domain folder (domain-{folder-name})
- **runtimeFields** — every domain and field slug must exist in the knowledge/ directory
- **Defaults** — every field config.json default value must match an existing option file slug
- **Package.json** — every space must have name, version, and private: true

For each broken reference, provide:
- The file containing the broken reference
- The exact broken value
- What it should reference (if determinable)
- A suggested fix

A single broken reference can cause runtime errors, so this validation must be thorough. Report zero issues only if every reference has been verified.
