# Deploy to Store

Submit your listing and make it live on the store.

**For free and source purchase:** Package the agent files (config.json, instruct.md, knowledge, flows) and submit. The store validates the package structure, displays your listing, and handles downloads.

**For API access:** First deploy the agent to a Fly.io space (use space-deploy). Then link the running deployment to your store listing. The store routes API requests to your space and handles billing.

After submission, verify the listing appears correctly:
- Description renders properly with formatting intact
- Pricing is displayed accurately
- Preview/demo works if you included one
- Test the purchase/download flow from a buyer's perspective

Monitor your listing's performance — downloads, revenue, and user feedback. Use this data to iterate on your description and pricing over time.
