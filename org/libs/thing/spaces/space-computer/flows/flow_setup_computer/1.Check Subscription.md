# Check Subscription

Verify the user has an active Computer tier Stripe subscription. The computer requires a paid subscription — without it, provisioning will fail.

Check via the get-usage endpoint or guide the user to the billing portal. If they're not subscribed, walk them through creating a checkout session with the Computer tier price_id via create-checkout.

Explain what the Computer tier includes: everything in Pro (full model catalog, higher token limits) plus a dedicated Fly.io machine with terminal access, 1 GB persistent storage, and start/stop control.
