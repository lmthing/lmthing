# Starter Tier

The Starter tier is an affordable entry point for users who need more than the free tier but aren't ready for a full Basic subscription. At $5/month, it provides a comfortable token budget for personal projects and experimentation.

## What's Included

- **Basic models** — Access to cost-efficient models like gpt-5.4-nano
- **$5/month token budget** — 5x the free tier's weekly allowance, resetting monthly
- **Studio access** — Full agent builder with all workspace features
- **Chat access** — Personal THING conversations with available models
- **GitHub sync** — Workspace persistence via your private GitHub repo
- **Store browsing** — Download free agents from the marketplace

## Limitations

- No access to premium models (GPT-4o, Claude Opus, Gemini Pro)
- Token budget resets monthly — when exhausted, wait for next period or upgrade
- No Computer node access (requires Computer tier)
- Rate limits: 25K tokens/min, 150 requests/min

## Who It's For

The Starter tier is ideal for hobbyists and personal projects that need consistent access without breaking the bank. It removes the weekly budget constraint of the free tier and gives you a predictable monthly allowance.

## Upgrading

When you need higher rate limits or a larger budget, consider the Basic tier ($10/month) or Pro tier ($20/month) for premium model access.
