# Apply Techniques

Based on the diagnosis, apply specific fixes:

- **Off-topic** → Strengthen boundaries. Add explicit "You do NOT" sections. Narrow enabledKnowledgeFields.
- **Too verbose** → Add conciseness constraints ("respond in under 200 words"). Lower temperature. Add format requirements.
- **Too generic** → Add knowledge-referencing instructions. Inject specific domain expertise. Use more specific identity framing.
- **Missing context** → Check runtimeFields — are required fields actually being populated? Add more knowledge domains if needed.
- **Wrong tool usage** → Add explicit tool guidance section. Specify when to use each tool and when not to.
- **Inconsistent tone** → Strengthen communication style section. Add format examples showing expected response structure.
- **Hallucination** → Add constraint: "only reference information from your knowledge domains." Switch to a more factual model.

Make one change at a time when possible. Multiple simultaneous changes make it hard to know which fix worked.
