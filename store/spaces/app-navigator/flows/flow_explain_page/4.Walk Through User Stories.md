# Walk Through User Stories

Walk through the key user stories for the page. Each story follows the format:

**As a** [user type]
**I want to** [do something]
**So that** [benefit/goal]

For the selected page, cover these primary stories:

### Home Page
- US-001: Understand platform value proposition
- US-002: Authenticate with GitHub
- US-004: Access Studio via workspace modal
- US-005/006: Explore and open demo workspaces

### Studio Dashboard
- US-101: See current workspace context
- US-102: Switch between workspaces
- US-104: Navigate via sidebar (Knowledge, Agents, Conversations)
- US-105/106: View Knowledge Areas and Agents overview
- US-107/108: Create new Knowledge Area or Agent
- US-112: Open Thing assistant panel
- US-113: Manage workspace settings

### Agent Configuration
- US-201: Navigate back and see agent identity
- US-202: Toggle knowledge pill attachments
- US-203: Edit main instructions
- US-205–207: View, attach, and manage flows/actions
- US-208: Test agent via Chat sandbox

### Knowledge Area Details
- US-301: Navigate back and see area context
- US-302–304: Browse, search, expand/collapse file tree
- US-305/306: Create new files and folders
- US-307/308: Select and edit file content
- US-310–313: Context menu: rename, duplicate, delete

For each story, briefly describe what the user sees and what happens when they interact.
