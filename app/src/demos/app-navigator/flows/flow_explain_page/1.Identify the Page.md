# Identify the Page

Ask the user which page they want to explore. The lmthing platform has three main page areas:

- **Home Page** (`/`) — The public-facing landing page with workspace selection
- **Studio Dashboard** (`/workspace/[workspaceId]/studio`) — The main workspace hub with Knowledge and Agents overviews
- **Agent Configuration** (`/workspace/[workspaceId]/studio/agent/[agentId]`) — Configure a specific AI agent
- **Knowledge Area Details** (`/workspace/[workspaceId]/studio/domain/[domainId]`) — Explore and edit knowledge files

Ask: "Which page would you like to understand? You can describe it by name or by the URL path."
