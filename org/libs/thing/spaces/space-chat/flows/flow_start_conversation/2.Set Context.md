Gather and structure the user's context into an effective context block for the conversation. Extract the following from the user's input:

- What they are working on (project, problem, task)
- Relevant background (domain expertise, constraints, preferences)
- What they have already tried (if troubleshooting)
- Desired output format (prose, bullet points, code)
- Their expertise level so the agent can calibrate explanations

Synthesize a concise, well-organized context block that maximizes response quality. More specific context produces better responses — refine vague inputs into concrete details.
