# Pro Tier

The Pro tier unlocks the full model catalog and higher token limits for serious agent building and daily use.

## What's Included

- **Full model catalog** — All available models across OpenAI, Anthropic, and Google
- **Premium models** — GPT-4o, GPT-4.1, o3, Claude Opus 4, Claude Sonnet 4, Gemini 2.5 Pro
- **Higher token limits** — Significantly more tokens per billing period
- **All Studio features** — Everything in free, plus premium model access
- **Full Chat access** — Use any model in personal conversations
- **Full Blog access** — Unlimited RSS feeds, full synthesis and research capabilities
- **Store publishing** — Publish agents to the marketplace with any distribution model

## How Billing Works

All LLM requests are metered through Stripe Token Billing. Stripe automatically counts input and output tokens, applies pricing based on the model used, and bills your account. You can check your balance via the get-usage endpoint or billing portal.

## Who It's For

Pro is for users who build agents regularly, need premium models for quality-sensitive tasks, or use lmthing as a daily tool. The higher token limits remove friction from iterative development and testing.

## Upgrading to Computer

If you need a persistent runtime environment with terminal access, consider the Computer tier which includes everything in Pro plus a dedicated Fly.io machine.
