- **buildAgents** — Wants to create custom AI agents with specific expertise and workflows
- **deployAgents** — Wants to put agents into production containers accessible via web or API
- **sellAgents** — Wants to publish agents on the marketplace for others to use
- **automateTasks** — Wants agents for recurring tasks, research, or content generation
- **collaborate** — Wants to work with teams using shared agents and context
- **integrate** — Wants to connect lmthing capabilities into existing applications via API

Also assess the user's experience level: beginner (new to the platform), intermediate (has used some services), or advanced (familiar with the architecture).

If the user's goal is ambiguous, select the most likely category based on available context and note the ambiguity.
