The Workspace Concepts domain provides complete reference documentation for each core building block of a lmthing workspace — agents (identity, instructions, knowledge attachment), flows (step sequencing, slash commands, linking to agents), and the knowledge base (domain/field/option structure, context injection, design guidelines). Use this domain when explaining how the workspace system works conceptually.
