# lmthing.store

The agent marketplace where creators publish and users discover AI agents. Three distribution models support different business strategies.

## What It Does

Store is the marketplace layer of lmthing. Creators list their agents for download or API access, set pricing, and earn revenue. Users browse, evaluate, and acquire agents to add to their own workspaces or call via API.

## Key Features

- **Three distribution models** — Free (download), source purchase (one-time payment), API access (per-token billing)
- **Agent discovery** — Browse, search, and filter agents by category and capability
- **Creator dashboard** — Track downloads, revenue, and user feedback
- **Pricing flexibility** — Set one-time prices or per-token markup percentages
- **Instant download** — Source-based agents download directly into the buyer's workspace

## When to Use

Use Store when you want to share or monetize agents you've built in Studio. Whether you're contributing to the community (free), selling specialized tools (source purchase), or running an agent-as-a-service business (API access).

## How It Connects

Agents are built in Studio, optionally deployed via Space (for API access listings), and listed on Store. Revenue flows through Stripe. Downloaded agents integrate into the buyer's workspace alongside their existing agents and knowledge.
