# Access Terminal

Once the machine is running, set up terminal access:

1. Get an access token via the `issue-computer-token` endpoint — returns a 5-minute HMAC-signed token
2. Connect to the terminal through the computer dashboard (xterm.js frontend)
3. The dashboard handles the WebSocket connection and token authentication automatically

If connecting programmatically, establish a WebSocket connection to `{fly_app_name}.fly.dev` with the token in the request. The token must be valid (within 5-minute TTL) and properly HMAC-signed.

Explain that the terminal provides full shell access — they can run commands, install packages, edit files, and manage processes just like a local terminal.
