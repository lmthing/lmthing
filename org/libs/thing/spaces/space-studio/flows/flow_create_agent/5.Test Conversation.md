# Test Conversation

Start a conversation with your new agent and verify it works correctly:

**Check these behaviors:**
- Does the agent stay in role? Ask it something outside its expertise — it should decline gracefully.
- Does it use knowledge correctly? Ask about a topic covered by its knowledge domains.
- Do slash actions work? Invoke each slash action and verify it triggers the linked flow.
- Is the tone right? Check that communication style matches the prompt's intent.
- Are tool calls appropriate? If the agent has edit access, does it read before editing?

**Iterate based on results:**
- If responses are too generic → add more specific instructions or knowledge references
- If responses are too verbose → add "be concise" constraints or lower temperature
- If the agent goes off-topic → strengthen the "do NOT" section
- If tool usage is wrong → add explicit tool guidance in the prompt
- If quality varies → try a different model (Claude for writing, GPT-4o for tools)

Test with multiple conversation scenarios before considering the agent ready.
