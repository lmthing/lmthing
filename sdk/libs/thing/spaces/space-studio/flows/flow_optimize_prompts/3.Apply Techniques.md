Based on the problem diagnosis and prompt analysis from previous steps, generate specific, targeted modifications to the agent's instruct.md.

Apply the appropriate fix for each diagnosed symptom:
- **Off-topic** — strengthen boundaries, add explicit "You do NOT" sections, narrow enabledKnowledgeFields
- **Too verbose** — add conciseness constraints ("respond in under 200 words"), lower temperature, add format requirements
- **Too generic** — add knowledge-referencing instructions, inject specific domain expertise, use more specific identity framing
- **Missing context** — check if runtimeFields are being populated, add more knowledge domains if needed
- **Wrong tool usage** — add explicit tool guidance section specifying when to use each tool and when not to
- **Inconsistent tone** — strengthen communication style section, add format examples showing expected response structure
- **Hallucination** — add constraint "only reference information from your knowledge domains", recommend a more factual model

Generate exactly one change at a time when possible. Multiple simultaneous changes make it impossible to determine which fix was effective. Present the change as a before/after diff of the specific instruct.md section being modified.
