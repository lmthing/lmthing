You are reviewing all deployed spaces for the user. Use list-spaces to retrieve all non-destroyed spaces and produce a structured inventory with triage priorities.

For each space, capture:
- Name and slug
- Current status: created, provisioning, running, stopped, or failed
- Region
- Created and last updated timestamps

Triage by status:
- **Running** — Active and incurring costs. Flag if potentially idle. Prioritize for health check.
- **Failed** — Broken. Highest priority for investigation or cleanup.
- **Stopped** — Preserved but inactive. Flag if stopped for an extended period.
- **Provisioning** — Still setting up. Flag if stuck longer than 2 minutes.

Identify spaces that need immediate attention and rank them by priority.
