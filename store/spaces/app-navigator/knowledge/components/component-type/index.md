The component type groups UI components into high-level categories: Navigation (header, sidebar, tree explorer), Content (card grid, empty state, pill bar, markdown editor), and Overlay (modal dialogs, floating action button, sliding panel). Selecting a type focuses agent explanations on that category of components.
