# Step 2: Environmental Setup

**Agent:** Behavior Trainer

## Task

Design environmental modifications to support behavior change.

## Inputs

- Behavior analysis from Step 1
- Home environment details
- Available resources and budget
- Owner's willingness to make changes

## Instructions

Create an environmental modification plan:

### 1. Prevent Unwanted Behavior
Make the unwanted behavior harder or less rewarding:
- Physical barriers
- Deterrents (cat-safe)
- Remove triggers where possible
- Change routines that trigger behavior

### 2. Encourage Alternative Behavior
Make desired behavior easier and more rewarding:
- Provide appropriate outlets
- Strategic resource placement
- Create appealing alternatives
- Increase enrichment opportunities

### 3. Resource Assessment
For each issue, ensure adequate:
- Litter boxes (number, location, cleanliness)
- Food/water stations
- Scratching surfaces
- Vertical territory
- Hiding spots
- Play opportunities
- Rest areas

### 4. Multi-Cat Considerations
If applicable:
- Separate resources
- Multiple access points
- Vertical territory for hierarchy
- Individual attention time

### 5. Implementation Priority
Rank changes by:
1. Safety/urgency
2. Quick wins (easy, high impact)
3. Longer-term modifications

## Output

A detailed environmental modification checklist with:
- Specific changes to make
- Shopping list (if needed)
- Setup instructions
- Before you start training notation
