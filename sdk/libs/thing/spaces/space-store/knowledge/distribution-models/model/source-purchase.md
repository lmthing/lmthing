# Source Purchase Distribution

Sell the complete source of your agent as a one-time purchase. Buyers get the full agent package downloaded into their workspace, where they can customize and extend it freely.

## When to Use

- Specialized, high-value agents that solve specific problems
- Industry-specific workflows with deep domain knowledge
- Complex multi-agent systems that took significant effort to build
- Agents with extensive, curated knowledge bases
- When you want simple, predictable revenue per sale

## How It Works

The buyer pays once and receives the complete agent: config.json, instruct.md, all knowledge domains and options, all flows, and values.json. The agent is downloaded directly into their workspace. They own it forever and can modify it without restrictions. You receive the purchase price minus the platform fee.

## Pricing Guidelines

Set a fair price based on the value delivered and the effort invested:

- **$5–$15** — Simple, focused agents with basic knowledge
- **$15–$50** — Specialized agents with curated knowledge bases
- **$50–$200** — Comprehensive multi-agent spaces with extensive flows

## Best Practices

- Offer a free preview or demo so buyers can evaluate before purchasing
- Include clear documentation explaining setup and configuration
- Keep the agent well-maintained — updates build trust for future purchases
- Consider offering a free "lite" version alongside the full paid agent
