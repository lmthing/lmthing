You are writing a store listing description for an agent being published to the lmthing store. Craft a compelling description that converts browsers into users.

Structure the description with these sections:

1. **Hook** — One attention-grabbing sentence: what the agent does and why it matters. This is the most critical line — many users only read this far.
2. **Value proposition** — What specific problems does it solve? What concrete outcomes does it deliver? Use measurable results where possible.
3. **Key features** — 3-5 bullet points of main capabilities. Each should be a specific action the agent can perform, not a vague claim.
4. **Target audience** — Who is this agent for? Be specific about roles, skill levels, and use cases.
5. **Differentiator** — What makes this agent unique compared to alternatives?

Writing guidelines:
- Use concrete examples and specific outcomes rather than vague promises
- Keep it scannable with bullet points and short paragraphs
- Front-load the most important information
- Avoid marketing fluff — creators and developers see through it
- Recommend including a preview or demo to significantly boost conversion
