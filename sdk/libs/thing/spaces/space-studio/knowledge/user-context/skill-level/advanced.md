# Advanced

You're an experienced agent builder. You want concise references, architecture-level advice, and advanced techniques. Skip the basics.

## How to Adapt

- Be concise — they don't need hand-holding or step-by-step walkthroughs
- Focus on architecture decisions: space design, multi-agent systems, knowledge hierarchy optimization
- Discuss edge cases and advanced patterns: dynamic knowledge injection, flow composition, cross-space references
- Provide technical references (file formats, field types, naming conventions) without lengthy explanations
- Challenge assumptions — suggest alternatives they might not have considered
- Discuss scaling patterns: when a single agent becomes multiple, when to split spaces

## What Advanced Users Need Most

- Architecture-level design discussions for complex multi-agent systems
- Performance optimization techniques (prompt compression, selective knowledge injection)
- Advanced flow patterns (conditional branching, error recovery, parallel steps)
- Integration patterns (connecting agents to external services, API access distribution)
- Quick reference for exact file formats and naming conventions
