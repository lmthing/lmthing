# Write Description

Craft a compelling listing description that converts browsers into users. Structure it as:

1. **Hook** — One attention-grabbing sentence: what the agent does and why it matters
2. **Value proposition** — What problems does it solve? What outcomes does it deliver?
3. **Key features** — 3-5 bullet points of main capabilities
4. **Target audience** — Who is this agent for? Be specific
5. **Differentiator** — What makes this agent unique?

Use concrete examples and specific outcomes rather than vague promises. Keep it scannable with bullet points and short paragraphs. Front-load the most important information — many users only read the first few lines.

Consider adding a preview or demo to show the agent in action. Listings with previews consistently outperform those without.
