# Choose Model

Help the user select the right model based on their task and subscription tier.

**Free tier users** get gpt-4o-mini and claude-3-5-haiku. Recommend based on the task: gpt-4o-mini for general questions and tool-heavy tasks, haiku for writing and concise analysis.

**Pro/Computer tier users** get the full catalog. Match model to task: GPT-4o for speed and versatility, Claude Sonnet for quality writing, o3 for complex reasoning, Gemini Pro for long documents.

Explain the tradeoffs clearly: speed vs capability, cost vs quality. Not every task needs a flagship model — encourage starting with efficient models and upgrading only when quality matters.
