Design flows (multi-step workflows) for the workspace and map each to the appropriate agent as a slash action. For each flow, generate a snake_case ID with flow_ prefix, a descriptive name, 4-7 numbered steps with clear titles, and the slash command that triggers it. Ensure each flow has a logical progression and that steps are discrete, actionable units.
