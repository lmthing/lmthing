# Identify Problem

Describe what's going wrong with the agent's conversations. Common symptoms:

- **Off-topic responses** — The agent answers questions outside its expertise
- **Too verbose** — Responses are unnecessarily long and include irrelevant information
- **Too generic** — Responses lack specificity and could come from any LLM
- **Missing context** — The agent doesn't use its knowledge domains effectively
- **Wrong tool usage** — The agent uses tools incorrectly or at wrong times
- **Inconsistent tone** — The agent's communication style varies between responses
- **Hallucination** — The agent makes up information not in its knowledge base

Understanding the symptom guides the diagnosis. Ask for specific examples of bad responses — concrete cases are much more useful than vague complaints like "it's not good enough."
