# Highlight Interactive Elements

For the selected page, summarize all interactive elements and what they trigger:

## Modal-triggering Actions
- Workspace selector dropdown → opens workspace list modal
- "+ Create Knowledge" / "+ Create Agent" → opens simple form modal
- Settings link → opens tabbed settings modal (package.json / Env Files / Providers)
- "Attach Flow" button (Agent page) → opens workflow selection modal
- "Open Studio →" (Home page) → opens demo workspace selection modal

## Navigation Actions
- Knowledge area card click → navigates to `/workspace/.../studio/domain/[domainId]`
- Agent card click → navigates to `/workspace/.../studio/agent/[agentId]`
- Back arrow in header → returns to `/workspace/.../studio`
- Sidebar list items → navigate between knowledge areas or agents

## Toggle/Panel Actions
- "Thing" button → slides in the right panel (C10), changes to "Hide Thing"
- "Collapse" button in sidebar → collapses sidebar to icon-only view
- Knowledge pills (Agent page) → toggles attach/detach state (gray ↔ violet)
- File tree folders → expand / collapse nested contents
- Action card status badge → toggles Active ↔ Inactive

## Contextual Menus
- Hover over file/folder in tree → reveals three-dot menu (rename, duplicate, delete)
- Hover over action card (Agent page) → reveals edit/disable/detach menu

## Editor & Chat
- "Chat" FAB (Agent page) → opens Runtime Conversation Preview overlay
- Clicking a file in tree (Knowledge page) → loads content in markdown editor

Explain how each interaction is visually signalled (hover shadows, color changes, icon transitions).
