# Home Page (`/`)

**Layout:** L1 – Single-Column Centered

The home page is the first thing users see. It presents the platform's value proposition and gives access to demo workspaces.

## Key Sections
- **Hero** — Headline "Turn Knowledge into AI Agents", tagline, and login button
- **Studio CTA Card** — "Build, test, and run everything in one place" with three feature pills
- **Demo Workspace Grid** — Three cards: local/education, local/plants, local/web-development

## Navigation Paths
- Click "Open Studio →" → opens "Select Demo Workspace" modal
- Click a demo workspace card "Open" button → navigates to `/workspace/{id}/studio`
- Click "Login with GitHub" → initiates GitHub OAuth

## Color Scheme
- Background: Off-white / light gray (slate-50) with subtle blue glow
- Primary CTA: Near-black slate
- Accent colors: Green, Purple, Orange for demo workspace icons
