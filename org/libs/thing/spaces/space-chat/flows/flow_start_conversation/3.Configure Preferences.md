# Configure Preferences

Set the conversation mode and any model-specific settings:

**Choose a mode:**
- **Casual** — Relaxed, exploratory. Good for brainstorming. Temperature: 0.7–0.8.
- **Focused** — Task-oriented, structured. Good for specific questions. Temperature: 0.3–0.5.
- **Creative** — Imaginative, bold. Good for writing and ideation. Temperature: 0.8–1.0.

Each mode shapes how the agent responds — make sure the mode matches your intent. Mismatched modes (creative mode for debugging) lead to frustrating interactions.

**Additional settings:**
- Temperature can be fine-tuned within the mode's range
- Some models have specific strengths that interact with mode choice (Claude + creative, GPT-4o + focused)
