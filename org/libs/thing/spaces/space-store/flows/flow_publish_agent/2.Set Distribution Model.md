# Set Distribution Model

Choose how to distribute your agent on the store. Three options:

**Free** — Maximum reach, no revenue. Best for community building, reputation, and lead generation. Users download the full source into their workspace.

**Source Purchase** — One-time payment for full source. Best for specialized, high-value agents. Buyers own and can customize the agent. Set a price based on value and complexity ($5–$200 range).

**API Access** — Per-token billing with your markup. Best for proprietary agents you want to keep private. Users call your deployed agent via API. Requires deploying to a space first.

Consider your goals: reach vs revenue vs control. Many creators start with free to build credibility, then offer paid agents once they have a track record.
