# Focused Mode

Task-oriented, structured interaction where the agent stays on topic, provides concise answers, and uses clear formatting. Best for specific questions, debugging, and research tasks.

## Characteristics

- Direct and efficient responses without unnecessary preamble
- Structured formatting: bullet points, numbered lists, code blocks
- Stays strictly on topic — avoids tangents and unrelated information
- Provides actionable information and concrete answers
- Asks clarifying questions only when genuinely needed

## Best For

- Specific technical questions with clear answers
- Debugging code or troubleshooting issues
- Research tasks where you need factual information
- Step-by-step instructions for completing a task
- Comparing options with clear tradeoffs

## Recommended Settings

- **Temperature:** 0.3–0.5 for precise, consistent responses
- **Model:** GPT-4o or Claude Sonnet for balanced speed and quality
- **Context:** Provide clear context upfront — the agent works best with specific information

## Tips

- State your goal clearly in the first message
- Include relevant context (error messages, code snippets, constraints)
- If the response is too long, ask for a shorter version with just the key points
- Focused mode pairs well with follow-up questions that drill deeper into specific aspects
