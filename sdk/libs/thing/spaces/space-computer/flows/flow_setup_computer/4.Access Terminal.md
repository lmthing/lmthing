Once the machine is running, set up terminal access by generating an access token and establishing the connection.

Steps to execute:
1. Request an access token via the issue-computer-token endpoint — returns a 5-minute HMAC-signed token
2. Connect to the terminal through the computer dashboard (xterm.js frontend)
3. The dashboard handles the WebSocket connection and token authentication automatically

For programmatic access, establish a WebSocket connection to {fly_app_name}.fly.dev with the token in the request. The token must be valid (within 5-minute TTL) and properly HMAC-signed.

The terminal provides full shell access — commands, package installation, file editing, and process management.
